# Data Structure Programs (DSA)

This repository contains a comprehensive collection of **Data Structures and Algorithms (DSA)** programs implemented by me in **C++** and **Java**.

---

## 🧠 Topics Covered

### 🔁 Linear Data Structures
- Stack using Array
- Queue using Array
- Circular Queue using Array
- Stack using Singly Linked List
- Queue using Singly Linked List
- Circular Queue using Singly Linked List

### 🔗 Linked Lists
- Singly Linked List
- Doubly Linked List
- Circular Linked List

### 🔤 Expression Conversion & Evaluation
- Infix to Postfix
- Infix to Prefix
- Postfix to Infix
- Postfix to Prefix
- Prefix to Infix
- Prefix to Postfix

### 🔍 Searching Algorithms
- Linear Search
- Binary Search

### 📊 Sorting Algorithms
- Selection Sort
- Insertion Sort
- Bubble Sort
- Optimized Bubble Sort
- Merge Sort
- Quick Sort
- Counting Sort
- Radix Sort
- Shell Sort
